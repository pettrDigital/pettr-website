A few things we don’t cover
  • Air-conditioners — though we can investigate leaks near an air-con unit.
  • Shopping centres (e.g. Westfield) — centres use their own preferred trades via the building manager.
  • Solar panels & inverters.
  • Solar hot-water heaters.
  • Gas heaters — note this is different from gas hot-water systems and heat-pump hot water, which we do service.
  • Vehicles & caravans — these need an auto electrician.
A quick note on appliances
We install and connect appliances, but we don’t do internal appliance repairs — for some faults a local appliance technician is the better call:
  • Oven / cooker: we can supply & install a new oven, but an element / “not heating up” fault needs an appliance technician.
  • Washing machine: a fault/error or one that won’t pump out needs an appliance technician. If it’s the hose or connection at the back, we can repair or clear it.
